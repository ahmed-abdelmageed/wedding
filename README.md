# Wedding Invitation - Ahmed & Shahd

## Setup

To use your custom couple illustration:

1. Save your couple image as `images/couple.png`
2. Open `index.html` and find the line with the couple illustration
3. Replace the data URL with: `src="images/couple.png"`

## Features

- ✉️ **Animated envelope** - Click to open
- 💍 **Wax seal** - Breaks when opening
- 💐 **Floating petals** - Romantic background animation
- 📜 **Beautiful invitation card** - Centered on screen when open
- ❌ **Close button** - Click X to close and reopen anytime
- 📱 **Responsive** - Works on all devices

## How to Use

1. Open `index.html` in your browser
